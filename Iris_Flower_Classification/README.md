# IRIS FLOWER CLASSIFICATION

# Project Overview
The IRIS FLOWER CLASSIFICATION project is a supervised machine learning application that classifies Iris flowers into one of three species—**Setosa**, **Versicolor**, or **Virginica**—based on their sepal and petal measurements. It is one of the most popular beginner-friendly classification projects and demonstrates the complete machine learning workflow.

# Objective
The objective of this project is to develop a machine learning model that accurately predicts the species of an Iris flower using its physical measurements. The project focuses on understanding data classification, model training, and performance evaluation.

# Dataset
This project uses the **built-in Iris dataset** available in the Scikit-learn library.

# Dataset Details

* **Total Samples:** 150
* **Number of Classes:** 3
* **Features:**

  * Sepal Length (cm)
  * Sepal Width (cm)
  * Petal Length (cm)
  * Petal Width (cm)

# Target Classes

* Iris Setosa
* Iris Versicolor
* Iris Virginica

# Technologies Used

* Python
* NumPy
* Scikit-learn

# Machine Learning Algorithm
*Decision Tree Classifier*
    The Decision Tree algorithm learns patterns from the training data by creating decision rules based on feature values, making it simple, interpretable, and effective for classification tasks.

# Project Workflow

1. Import the required Python libraries.
2. Load the Iris dataset from Scikit-learn.
3. Explore the dataset and feature names.
4. Split the dataset into training and testing sets.
5. Train the Decision Tree Classifier.
6. Predict flower species using the trained model.
7. Evaluate the model using the Accuracy Score.
8. Test the model with new flower measurements.

# Results

The Decision Tree model successfully classified the Iris flower species with *high accuracy* on the test dataset, demonstrating its effectiveness for multiclass classification.
