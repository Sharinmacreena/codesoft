SALES PREDICTION USING PYTHON

# Project Overview
The **Sales Prediction Using Python** project is a machine learning application that predicts product sales based on advertising expenditures across different marketing channels. By analyzing the relationship between advertising budgets and sales performance, the model helps estimate future sales and supports data-driven marketing decisions.

# Objective
The primary objective of this project is to build a **Linear Regression** model that accurately predicts product sales using advertising investments in:

*  TV
*  Radio
*  Newspaper

# Dataset
The dataset contains the following features:

* TV Advertising Budget
* Radio Advertising Budget
* Newspaper Advertising Budget
* Sales Revenue (Target Variable)

# Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib (for visualization, if applicable)

# Machine Learning Algorithm
* **Linear Regression**

# Project Workflow

1. Import the required Python libraries.
2. Load and explore the sales dataset.
3. Perform data preprocessing and analysis.
4. Select the advertising budget features.
5. Split the dataset into training and testing sets.
6. Train the Linear Regression model.
7. Predict sales using the test dataset.
8. Evaluate the model's performance.

# Results
The Linear Regression model successfully predicted sales based on advertising expenditure across different media channels. The project demonstrates how machine learning can be applied to forecast sales and assist in making informed marketing decisions.

# Learning Outcomes

* Understanding regression-based machine learning.
* Data preprocessing and exploratory data analysis.
* Feature selection for predictive modeling.
* Building and training a Linear Regression model.
* Making predictions using unseen data.
* Evaluating model performance and interpreting results.

# Conclusion

This project highlights the effectiveness of Linear Regression in predicting sales from advertising investments. It provides valuable insights into how different advertising channels influence product sales and serves as a strong foundation for learning predictive analytics.
